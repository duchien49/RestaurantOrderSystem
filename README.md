# RestaurantOrderManagementSystem

## Description
Console-based C++ application for managing restaurant orders.

## Features
- Manage menu
- Create and process orders
- Order status management
- Checkout and payment

## Build & Run
- Compile all .cpp files
- Run main.cpp

## Architecture
- OOP-based
- Classes mapped from UML Class Diagram
